# Delphi-AHP Pipeline

德尔菲-层次分析法（Delphi-AHP）研究流程工具。帮助MBA和管理学研究生快速完成专家访谈、因子收敛分析和权重计算，生成结构化研究报告。

---

## ⚡ 一键安装

> 📂 已附带两套完整示例数据：
> - `sample/live_ecommerce_research/` — 直播电商主播信任研究
> - `sample/ev_charging_research/` — 新能源汽车充电站选址研究
>
> 克隆后可直接查看输出效果。

**macOS / Linux**（打开终端，输入以下命令回车）：

```bash
curl -fsSL https://raw.githubusercontent.com/stephenlzc/delphi-ahp-pipeline/main/install.sh | bash
```

**Windows**（打开 PowerShell，输入以下命令回车）：

```powershell
irm https://raw.githubusercontent.com/stephenlzc/delphi-ahp-pipeline/main/install.ps1 | iex
```

| 系统 | 推荐终端 |

## 📋 功能概览

| 步骤 | 内容 |
|:---:|:---|
| 1 | 输入研究主题、背景、目的 |
| 2 | 配置 LLM API（调用AI专家） |
| 3 | 添加专家小组 |
| 4 | 设置访谈轮次 |
| 5 | 自动运行 Delphi + AHP 分析 |
| 6 | 查看收敛结果 |
| 7 | 查看权重分析 |
| 8 | 生成研究报告 |

## ✨ 当前功能

### 🔄 断点续作

程序支持**中断后从断点继续**。如果在运行过程中退出（如关闭终端、断电等），下次启动时可以选择"继续上一次的研究"，程序会自动跳到中断的步骤，无需从头开始。

### 🤖 AI 专家生成

无需手动填写专家信息，只需输入专家数量，程序会自动调用 LLM 生成具有不同背景、专业领域和行为风格的专家档案。支持高校、企业、政府、MCN 等多种机构类型。

### 🔗 多 API 提供商支持

支持同时配置多个 LLM 提供商，可在 DeepSeek、Moonshot、Kimi、OpenAI 等之间灵活切换。不同专家可使用不同的 API 和模型。

### 🌐 德尔菲访谈语言可选

在步骤 1 中可选择专家访谈的语言（中文 / English）。程序会根据选择调整所有 Prompt 的输出语言指令，确保 AI 专家使用指定语言进行回答。

### 📊 收敛检验

程序自动计算每位专家对各因素打分的变异系数（CV）。当 CV ≤ 0.5 时，表明专家意见已收敛，结果可信。

### ✅ AHP 一致性检验

自动构建两两比较判断矩阵，计算权重和一致性比率（CR）。CR ≤ 0.1 时通过一致性检验，否则提示重新调整判断。

### 📁 多格式报告导出

支持导出多种格式：
- **Markdown** — 可直接粘贴到 Word 或笔记软件
- **HTML** — 可在浏览器中查看，带样式
- **Excel** — 包含 AHP 权重表、专家评分表等分析表格
- **Word** — 交互式 Word 文档格式

## 🗺️ 开发计划

本项目目前处于早期阶段（CLI 版本），以下是未来规划方向，欢迎贡献想法。

### 交互形式

- **网页版 UI**：开发可视化的 Web 界面，替代命令行，通过浏览器引导用户逐步完成 8 个步骤，降低使用门槛
- **进度保存与恢复**：支持在任意步骤中断后从断点继续，而非从头开始
- **多语言界面**：界面支持中文、English 等多种语言

### AI 集成

- **更多模型支持**：聚合更多 LLM 提供商（如 Kimi、Qwen、Claude 等），用户可灵活切换
- **模型对比分析**：同一位专家使用不同模型回答，对比分析结果的差异
- **自动优化 Prompt**：根据用户选择的研究领域，自动调整 Prompt 措辞和专业术语
- **AI 辅助撰写**：在报告生成阶段，调用 LLM 对原始分析结果进行润色和深化

### 技术优化

- **Token 节省方案**：研究并实施 Prompt 压缩、历史摘要等技术，降低 API 调用成本
- **缓存机制**：对相同结构的 Prompt 进行缓存，避免重复调用
- **增量执行**：已完成的步骤可跳过执行，节省时间和费用
- **离线模式**：内置模拟 LLM 响应，支持无网络环境下演示和测试

### 数据与分析

- **可视化图表**：将 AHP 权重、收敛趋势等结果以图表形式展示（柱状图、雷达图等）
- **Excel/Word 导出**：一键导出符合学术规范的表格和文档
- **历史项目管理**：支持查看、对比、复用历史研究项目
- **PDF 报告**：生成可直接提交的 PDF 格式研究报告

### 协作与分享

- **专家评审功能**：支持多人协作，每位专家独立完成访谈
- **研究模板库**：提供常见研究领域的访谈框架模板（如 PEST、SWOT、市场进入等）
- **一键复现**：通过配置文件分享研究设置，方便导师或同伴复现分析过程

### 稳定性与体验

- **自动重试机制**：网络波动时自动重试，提升大规模调用稳定性
- **详细日志**：记录每次 LLM 调用的输入输出，便于审计和调试
- **错误诊断**：遇到 API 错误时给出具体原因和解决建议
